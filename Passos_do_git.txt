Comandos de utilização git bash 

git clone <link>
cd <caminho> 
cd // sem o caminho, ele retorna o caminho
ls // arquivos localizados no diretorio
Após acessar o diretorio via master: 
vi <nome do arquivo>
git status // checa se o arquivo está sendo utilizado ou não, se não utilizar comando git add * e então utilizar status novamente
git add * //especifique o arquivo, ou utilize * para todos os arquivos 
git commit -m "<comentar seu nome e suas alterações do dia>"
git push origin master // Colocar seu login e senha do github OBS: a senha ficara invisivel
